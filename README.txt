SUMMARY
-------

The Sympa Subscribe module provides a block for subscribing to a Sympa mailing
list.


REQUIREMENTS
------------

None.


INSTALLATION
------------

Install as usual.  See http://drupal.org/documentation/install/modules-themes
for more information.


CONFIGURATION
-------------

Customize Sympa mailing list and block appearance in:

  Administration -> Configuration -> System -> Sympa Subscribe

Add block in:

  Administration -> Structure -> Blocks
